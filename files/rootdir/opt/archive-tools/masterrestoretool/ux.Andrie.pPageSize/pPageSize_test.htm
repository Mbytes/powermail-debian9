  • beforeText: string - text to display before the ComboBox (defaults to 'Show')
  • afterText: string - text to display after the ComboBox (defaults to 'items')
  • addBefore: mixed - what to add before the combo (defaults to '-')
  • addAfter: mixed - what to add after the combo (defaults to null)
  • variations: array - what to substract or add to the selected value, in order to determine new pageSize options (defaults to [5, 10, 20, 50, 100])
  • dynamic: bool - make variations dynamic (defaults to false)
  • comboCfg: object - config object for the combo component
Methods
  • changePageSize(pageSize) - change pageSize
Methods added to the PagingToolbar instance
  • setPageSize(pageSize) - set pageSize
  • hetPageSize - get pageSize
